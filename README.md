# Capstone Project -IBM Data Science Professional Certificate

Data Science using Python to Collect, Analyze, Visualize Data, Build Machine Learning Model and Present Results using SpaceX Falcon 9 Launches Data

## 📌 Project Outline

1. Collect data using API and web scraping
2. Data wrangling using Pandas and NumPy
3. Perform exploratory data analysis using SQL
4. Visualize data using Matplotlib, Seaborn, Plotly, Folium and Dash
5. Build machine learning models using scikit-learn
6. Present results using Microsoft Powerpoint

## 🛠️ Tech Stack & Skills Covered

- Data: SpaceX Falcon 9 Launches Data
- Languages: Python
- Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn, Folium, Plotly and Dash
- Key Concepts: Data Wrangling, Exploratory Data Analysis (EDA), Predictive Modeling (Logistic Regression, SVM, Decision Trees, KNN), Hyperparameter Tuning
- Tools: Jupyter Notebook, VS Code, Microsoft Powerpoint
- Skills: Data Cleaning, Data Exploration and Analysis, Data Visualization, Machine Learning
